# About Dev.to Web Extension

This is a: 
1. passion project. 
2. web extension aiming to improve [dev.to](https://dev.to/) reading experience by hiding distracting UI elements and adding a custom read aloud feature. 

# Why this exists?

My passion for engineering in general was dying. Part of what coding fun or at least the idea of it for me was dying. 

I don't want to lose this part of me. So this project serves as the first small step towards it. 

With my passion projects I plan on not worrying about interviews, over-engineering or imposter syndrome in general. Just building what I like, the way I like.
